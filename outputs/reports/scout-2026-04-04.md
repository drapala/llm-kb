## Candidatos a ingestão (rankeados por impacto)

### 1. Graph-based Agent Memory: Taxonomy, Techniques, and Applications (Feb 2026)
- URL: https://arxiv.org/abs/2602.05665
- Relevante para: [[agent-memory-architectures]], [[reflexion-weighted-knowledge-graphs]]
- Impacto: **REFINA** — comprehensive taxonomy covering memory lifecycle (extraction, storage, retrieval, evolution). Includes topology optimization where agents increase edge weights — directly covers RWKG's proposed mechanism.
- Justificativa: Most comprehensive survey on graph-based memory (2026). May subsume multiple existing articles' claims under one authoritative source.
- **PRIORIDADE: ALTA** — could restructure agent-memory-architectures significantly.

### 2. How Reliable are LLMs as Knowledge Bases? (Jul 2024)
- URL: https://arxiv.org/abs/2407.13578
- Relevante para: [[autonomous-kb-failure-modes]], [[llm-as-judge]]
- Impacto: **REFINA** — introduces factuality + consistency metrics for LLMs as KBs. Tests 26 models. Finds "larger models give more consistent WRONG responses" — directly relevant to our authority cascade failure mode.
- Justificativa: Provides empirical grounding for failure modes we described as speculative. UnseenQA benchmark could validate our confidence scoring system.
- **PRIORIDADE: ALTA** — could move autonomous-kb-failure-modes from "speculative" to "empirically grounded."

### 3. A-MEM: Agentic Memory for LLM Agents (Feb 2025)
- URL: https://arxiv.org/abs/2502.12110
- Relevante para: [[agent-memory-architectures]], [[memory-consolidation]]
- Impacto: **REFINA** — Zettelkasten-style dynamic indexing + memory evolution (new memories trigger updates to existing ones). Directly parallels our wiki /review pattern. Tested on 6 foundation models.
- Justificativa: A-MEM's "memory evolution" where new memories update existing ones = our /ingest updating existing articles. Academic validation of our pattern.
- **PRIORIDADE: MÉDIA** — validates existing approach more than challenges it.
