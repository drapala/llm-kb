# CoALA: Cognitive Architectures for Language Agents

## Abstract

Proposes CoALA framework for understanding language agents with modular memory components, structured action space for internal memory and external environments, and generalized decision-making process. Surveys recent developments, identifies directions toward more capable agents. Positions language agents within AI's broader history of cognitive science and symbolic AI.

## Key Contribution

Provides the bridge between classical cognitive architectures (SOAR, ACT-R) and modern LLM agents. Formalizes what "memory" means for a language agent vs. what it meant in traditional cognitive science.

## Relevance to KB

CoALA's formalization of agent memory types (working, episodic, semantic, procedural) from cognitive science creates direct tension with our engineering-first taxonomy (factual, experiential, working from the Memory Survey). The cognitive science categories have decades of experimental validation; the engineering categories are pragmatic but ungrounded.
